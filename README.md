# HyperProof Lab

> "AI가 문제를 푼다. 인간은 문제를 정의한다."

HyperProof Lab은 AI의 한계를 매일 밀어붙이는 사람들입니다.

---

## 📁 폴더 구조

```
hypeproof/
├── docs/                    # 📚 운영 문서 (OS 등)
├── members/                 # 👥 멤버별 폴더
├── content/                 # 🎙️ 트랙 A: 콘텐츠 (브랜딩)
│   ├── seasons/            # 시즌별 에피소드
│   ├── assets/             # 오디오, 숏폼, 썸네일
│   └── topic_pool.md       # 주제 풀
├── education/               # 📖 트랙 B: 교육
├── research/                # 🔬 리서치 시스템
│   ├── claims/             # Claim 카드 아카이브
│   └── lens/               # HypeProof Lens 가이드
├── action_items/            # ✅ 주간 Action Items
└── templates/               # 📋 템플릿 모음
```

## 🚀 빠른 시작

### 1. 새 에피소드 준비

```bash
# templates에서 복사
cp templates/episode_prep.md content/seasons/season_1/ep0X/prep.md
```

### 2. Claim 카드 작성

```bash
# templates에서 복사
cp templates/claim_card.md research/claims/2026-01/claim_XXX.md
```

### 3. 주간 Action Items

```bash
# templates에서 복사
cp templates/action_item_weekly.md action_items/2026/01_january/week_XX.md
```

## ⚠️ Red Lines (금지 사항)

- 실명 고객사/파트너사 금지
- 내부 수치 범위화/비율화

- 화면 공유 시 내부 UI/데이터 노출 주의
